# Frontend - Streamlit

Streamlit web application for FPP Indonesia Job Search.

## Setup

```bash
# Install dependencies
pip install streamlit requests

# Run the app
streamlit run frontend/streamlit/app.py
```

The app will be available at: http://localhost:8501

## Features

- 🔍 Job Search
- 💬 Chat with AI
- 💡 Career Advice
- 📊 Market Statistics
- 💰 Salary Information

## Pages

1. **Home** - Introduction and feature overview
2. **Search Jobs** - Search for job listings
3. **Chat** - Chat with AI assistant
4. **Career Advice** - Get personalized career guidance
5. **Statistics** - View market statistics by location
